2021-nu

CLE 1, de Care Bot

Voor het eerste project binnen mijn opleiding hebben we een robot gemaakt, deze robot kan medicatie dispensen door het gebruik van een ronddraaiend rad. Ook kan deze robot water inschenken om de medicatie mee in te nemen. Het zou nog wel leuk zijn om de robot te kunnen automatiseren doormiddel van een app zodat je bijvoorbeeld een schema zou kunnen invullen.

Ik was de doelgroep van deze robot omdat ik last heb van chronnische migraines en deze robot me elke ochtend zou herrineren om mijn medicijnen te nemen Bij deze robot heb ik het idee bedacht van het rad en het 3d model van deze robot gemaakt en geprint. Hierbij heb ik deze robot ook geverfd. Bij de code heb ik wel hier en daar geholpen maar bij dit project was het meer voormezelf om te werken naar het design aspect.
